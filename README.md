## Running locally

Clone the project

```bash
  git clone https://github.com/erarich/projeto_de_sistemas.git
```

Enter the directory project

```bash
  cd conhecimentoEmAcao
```

Install all dependencies
```bash
  npm install
```

Start dev server

```bash
  npm run dev (COrrigir)
```
